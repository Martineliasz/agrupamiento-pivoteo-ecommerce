Proyecto: Agrupamiento y pivoteo de datos con Pandas

En este proyecto se aplican técnicas de agrupamiento, pivoteo y combinación de datos
utilizando la librería Pandas en Python, a partir de un dataset real de e-commerce.

El objetivo es reorganizar y transformar la información para facilitar su análisis,
siguiendo los contenidos vistos en la clase de Agrupamiento y Pivoteo de Datos.

Dataset
El dataset utilizado contiene información de transacciones de un comercio electrónico,
incluyendo datos como número de factura, productos, cantidades, precios, clientes y país.
Los datos presentan características reales como valores nulos, registros repetidos y
múltiples productos por factura.

Actividades realizadas
- Carga y exploración del dataset.
- Revisión de tipos de datos y valores nulos.
- Eliminación de registros duplicados.
- Creación de métricas derivadas (total por línea de venta).
- Agrupamiento de datos con groupby y funciones agregadas.
- Aplicación de funciones personalizadas con agg y nunique.
- Pivotado de datos con pivot_table para análisis por país y período.
- Despivotado de datos utilizando melt.
- Combinación de DataFrames mediante concat.
- Fusión de DataFrames con merge utilizando distintos tipos de join.
- Exportación de los resultados a archivos CSV y Excel.

Herramientas utilizadas
- Python
- Pandas
- Visual Studio Code

Archivo principal
